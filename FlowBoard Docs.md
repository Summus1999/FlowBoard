## 产品概述

FlowBoard 是一款面向开发者和个人用户的桌面工作台应用，采用 Electron + Web 技术栈构建，提供 glassmorphism（毛玻璃）设计风格的现代化界面。产品整合了账户管理、学习追踪、日程规划、技术刷题、应用启动等多功能于一体，帮助用户高效管理数字生活。

### 设计理念

- **一站式管理**：减少在不同应用间切换的成本
- **简洁高效**：保留核心功能，避免功能冗余
- **数据本地化**：优先使用 localStorage 存储，保护隐私
- **开发者友好**：针对程序员场景优化（LeetCode、GitHub 集成）
- **智能辅助**：代码智能提示，提升刷题效率

## 核心功能详解

### 1. 我的主页

**入口**：侧边栏「我的主页」或底部 🏠 房子按钮

**功能组件**：

|组件|功能描述|交互方式|
|-----|--------|--------|
|账户安全概览|显示密码管理入口|点击「查看全部」进入账户管理页面|
|待办事项|显示全部待办任务（支持滚动浏览）|点击复选框完成/取消，点击 × 删除，点击「添加待办」新增|
|热榜资讯|显示热门新闻头条|点击跳转外部链接|

**交互逻辑**：

- 主页是应用启动后的默认页面
- 所有模块数据与其他页面实时同步
- 待办事项仅在主页显示，移除独立页面入口

### 2. 账户密码管理

**入口**：主页「查看全部」或独立的账户管理页面

**预设账户**（12个国内常用账号）：
微信、支付宝、LeetCode 中国、QQ、微博、淘宝、京东、Bilibili、网易云音乐、百度、GitHub、抖音

**功能特性**：

- **分类管理**：社交/金融/工作/娱乐
- **CRUD 操作**：
  - 添加：点击「添加账户」→ 填写平台/用户名/密码/分类 → 保存
  - 编辑：点击卡片上的编辑图标 → 修改信息 → 保存
  - 删除：点击删除图标 → 确认删除
- **密码强度检测**：添加时自动计算（弱/中/强）
- **快捷操作**：点击复制图标可复制用户名/密码

**交互逻辑**：

1. 添加账户时根据分类自动分配图标和颜色
2. 密码显示/隐藏切换（眼睛图标）
3. 分类筛选按钮可快速过滤

### 4. 待办事项

**入口**：主页中部「待办事项」卡片

**功能特性**：

- **添加待办**：点击「添加待办」→ 输入内容 → 选择标签（紧急/重要/一般/日常）→ 确认

**标签类型**：

- 🔴 紧急（红色）
- 🟠 重要（橙色）
- 🔵 一般（蓝色）
- 🟢 日常（绿色）

**注意**：待办事项已移除独立的侧边栏入口，统一整合到主页显示。

### 8. 面试追踪

**入口**：侧边栏「面试追踪」

**核心功能**：

#### 8.1 录音功能

- **开始录音**：点击红色「开始录音」按钮 → 请求麦克风权限 → 开始计时
- **停止录音**：点击「停止录音」按钮 → 保存录音文件
- **录音状态**：
  - 录音中：按钮变绿色，呼吸动画效果
  - 计时器：显示录制时长（分:秒）

#### 8.2 录音设置

- **存储路径**：自定义录音文件保存目录
- **录音质量**：高质量(320kbps)/标准(128kbps)/节省空间(64kbps)
- **自动保存**：录音结束后自动保存到指定路径
- **音乐暂停**：录音时自动暂停背景音乐

#### 8.3 录音管理

- **播放录音**：点击播放按钮在线回放
- **下载录音**：下载为 .webm 格式文件
- **删除录音**：从列表和存储中删除

#### 8.4 面试辅助

- **面试笔记**：文本域记录面试要点
- **面试 Checklist**：
  - 自我介绍准备
  - 项目经验梳理
  - 技术知识点复习
  - 提问准备
  - 代码演示准备

**交互逻辑**：

1. 首次使用请求麦克风权限
2. 录音使用 MediaRecorder API
3. 录音列表实时更新
4. 支持在面试过程中随时开始/停止录音

### 9. LeetCode 刷题

**入口**：侧边栏「LeetCode」

**功能模块**：

#### 9.1 题目列表

- **数据来源**：LeetCode API（优先）/ 本地模拟数据（离线）
- **筛选条件**：难度（简单/中等/困难）、状态（全部/已解决/未解决）
- **搜索**：按题目名称/编号搜索
- **数据标识**：☁️ 官方数据 / 💾 本地数据

#### 9.2 代码编辑器

- **编辑器**：Monaco Editor（VS Code 同款）
- **支持语言**：JavaScript/Python/Java/C++/TypeScript/Go/Rust 等
- **主题**：leetcode-dark 暗色主题
- **快捷键**：Ctrl+S 运行，Ctrl+Enter 提交

#### 9.3 代码智能提示 ⭐ 新增

- **算法模板**：两数之和、BFS、DFS、动态规划、二分查找、快速排序、并查集、优先队列、滑动窗口、回溯算法、LCA 等
- **代码片段**：for循环、foreach、map、filter、reduce 等常用代码块
- **触发方式**：输入前缀（如 `twosum`, `bfs`, `dp`）自动提示
- **多语言支持**：JavaScript/TypeScript、Python、Java、C++

#### 9.4 代码执行

- **运行代码**：本地模拟执行，显示控制台输出
- **提交代码**：模拟提交，显示通过/失败结果
- **控制台**：显示执行结果、用时、内存消耗

#### 9.5 登录功能

- Cookie 登录（支持 LeetCode 中国/国际版）
- 登录后获取真实题目数据和提交记录

#### 9.6 官网链接 ⭐ 更新

- **位置**：页面右上角「官网」按钮
- **样式**：圆角胶囊形状，渐变色背景，与主题风格匹配
- **交互**：悬停发光效果，点击跳转 LeetCode 官网

**交互逻辑**：

1. 点击题目列表加载题目详情
2. 切换语言时自动加载对应代码模板和智能提示
3. 支持代码重置（恢复模板）
4. 代码编辑器支持智能补全和语法高亮

### 10. GitHub 项目跟踪 ⭐ 更新

**入口**：侧边栏「GitHub」

**功能特性**：

- **用户登录**：支持 GitHub 用户名登录，可选 Personal Access Token（访问私有仓库）

  - 公开仓库数
  - 关注者数

6. 点击「官网」按钮访问 GitHub 官网

## 交互逻辑汇总

```javascript

                        ↓

```

```javascript
用户操作 → 修改 State → localStorage 持久化 → 重新渲染 UI
                ↓
            触发其他页面同步更新
```
