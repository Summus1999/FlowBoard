## 产品概述

FlowBoard 是一款面向开发者和个人用户的桌面工作台应用，采用 Electron + Web 技术栈构建，提供 glassmorphism（毛玻璃）设计风格的现代化界面。产品整合了账户管理、学习追踪、日程规划、技术刷题、应用启动等多功能于一体，帮助用户高效管理数字生活。

### 设计理念

- **一站式管理**：减少在不同应用间切换的成本
- **简洁高效**：保留核心功能，避免功能冗余
- **数据本地化**：优先使用 localStorage 存储，保护隐私
- **开发者友好**：针对程序员场景优化（LeetCode、GitHub 集成）
- **智能辅助**：代码智能提示，提升刷题效率

## 核心功能详解

### 1. 我的主页

**入口**：侧边栏「我的主页」或底部 🏠 房子按钮

**功能组件**：

| 组件 | 功能描述 | 交互方式 |
| ------ | ---------- | ---------- |
| 账户安全概览 | 显示密码管理入口，安全评分圆环 | 点击「查看全部」进入账户管理页面 |
| 待办事项 | 显示全部待办任务（支持滚动浏览） | 点击复选框完成/取消，点击 × 删除，点击「添加待办」新增 |
| 热榜资讯 | 显示热门新闻头条 | 点击跳转外部链接 |


**顶部统计栏**：

- 账户密码数量统计
- 待办任务数量统计（未完成的待办数）


**交互逻辑**：

- 主页是应用启动后的默认页面
- 所有模块数据与其他页面实时同步
- 待办事项仅在主页显示，移除独立页面入口

### 2. 账户密码管理

**入口**：主页「查看全部」或独立的账户管理页面

**预设账户**（12个国内常用账号）：
微信、支付宝、LeetCode 中国、QQ、微博、淘宝、京东、Bilibili、网易云音乐、百度、GitHub、抖音

**功能特性**：

- **分类管理**：社交/金融/工作/娱乐
- **CRUD 操作**：
  - 添加：点击「添加账户」→ 填写平台/用户名/密码/分类 → 保存
  - 编辑：点击卡片上的编辑图标 → 修改信息 → 保存
  - 删除：点击删除图标 → 确认删除
- **密码强度检测**：添加时自动计算（弱/中/强）
  - 评分规则：长度≥8位+1分，≥12位+1分，包含大小写+1分，包含数字+1分，包含特殊字符+1分
- **快捷操作**：点击复制图标可复制用户名/密码
- **安全评分**：主页显示安全评分圆环（基于密码强度计算）

**交互逻辑**：

1. 添加账户时根据分类自动分配图标和颜色
2. 密码显示/隐藏切换（眼睛图标）
3. 分类筛选按钮可快速过滤
4. 编辑模式支持修改已有账户信息

### 8. 面试追踪

**入口**：侧边栏「面试追踪」

**核心功能**：

#### 8.1 录音功能

- **开始录音**：点击红色「开始录音」按钮 → 请求麦克风权限 → 开始计时
- **停止录音**：点击「停止录音」按钮 → 保存录音文件
- **录音状态**：
  - 录音中：按钮变绿色，呼吸动画效果
  - 计时器：显示录制时长（分:秒）

#### 8.2 录音设置

- **存储路径**：使用 IndexedDB 存储录音文件
- **录音质量**：可调节比特率（高质量/标准/节省空间）
- **自动保存**：录音结束后自动保存

#### 8.3 录音管理

- **播放录音**：点击播放按钮在线回放
- **下载录音**：下载为 .webm 格式文件
- **删除录音**：从列表和存储中删除

#### 8.4 面试辅助

- **面试笔记**：文本域记录面试要点
- **面试 Checklist**：
  - 自我介绍准备
  - 项目经验梳理
  - 技术知识点复习
  - 提问准备
  - 代码演示准备

**交互逻辑**：

1. 首次使用请求麦克风权限
2. 录音使用 MediaRecorder API
3. 录音存储在 IndexedDB（支持大文件）
4. 录音列表实时更新

### 9. LeetCode 刷题

**入口**：侧边栏「LeetCode」

**功能模块**：

#### 9.1 题目列表

- **数据来源**：LeetCode API（优先）/ 本地模拟数据（离线）
- **筛选条件**：难度（简单/中等/困难）、状态（全部/已解决/未解决）
- **搜索**：按题目名称/编号搜索
- **数据标识**：☁️ 官方数据 / 💾 本地数据

#### 9.2 代码编辑器

- **编辑器**：Monaco Editor（VS Code 同款）/ 备用 textarea 编辑器
- **支持语言**：JavaScript/Python/Java/C++/TypeScript/Go/Rust 等
- **主题**：leetcode-dark 暗色主题
- **快捷键**：Ctrl+S 运行，Ctrl+Enter 提交
- **降级方案**：Monaco 加载失败时自动切换到 textarea 编辑器

#### 9.3 代码智能提示

- **算法模板**：两数之和、BFS、DFS、动态规划、二分查找、快速排序、并查集、优先队列、滑动窗口、回溯算法、LCA 等
- **代码片段**：for循环、foreach、map、filter、reduce 等常用代码块
- **触发方式**：输入前缀（如 `twosum`, `bfs`, `dp`）自动提示
- **多语言支持**：JavaScript/TypeScript、Python、Java、C++

#### 9.4 代码执行

- **运行代码**：本地模拟执行，显示控制台输出
- **提交代码**：模拟提交，显示通过/失败结果
- **控制台**：显示执行结果、用时、内存消耗

#### 9.5 提交历史

- 本地存储提交记录（最多500条）
- 统计已解决问题数
- 计算连续打卡天数

#### 9.6 登录功能

- Cookie 登录（支持 LeetCode 中国/国际版）
- 登录后获取真实题目数据和提交记录

#### 9.7 官网链接

- **位置**：页面右上角「官网」按钮
- **样式**：圆角胶囊形状，渐变色背景，与主题风格匹配
- **交互**：悬停发光效果，点击跳转 LeetCode 官网

**交互逻辑**：

1. 点击题目列表加载题目详情
2. 切换语言时自动加载对应代码模板和智能提示
3. 支持代码重置（恢复模板）
4. 代码编辑器支持智能补全和语法高亮

### 11. 应用中心

**入口**：侧边栏「应用中心」（原「提交记录」）

功能概述：快速启动本地安装的应用程序，支持图标 DIY 与桌面端拖拽排序

**应用分类**：

| 分类 | 包含应用 |
| ------ | --------- |
| 社交应用 | 微信、QQ、TIM、钉钉、飞书 |
| 开发工具 | VS Code、IntelliJ IDEA、WebStorm、DataGrip、Postman、Git Bash |
| 浏览器 | Chrome、Edge、Firefox |
| 媒体娱乐 | 网易云音乐、QQ音乐、哔哩哔哩 |
| 办公效率 | WPS Office、Typora、Notion、XMind、Snipaste |
| 系统工具 | 命令提示符、PowerShell、文件资源管理器、任务管理器 |

自定义应用：

- 添加应用：点击「添加应用」→ 输入名称 → 选择可执行文件路径 → 选择图标模式（预设/Emoji/上传图片）→ 保存
- 图标 DIY：提供 25 种预设图标，支持 Emoji 与本地图片上传
- 图片限制：仅支持 png/jpg/jpeg/webp，不支持 svg，上传后自动压缩
- 编辑/删除：支持修改应用信息和删除应用

排序管理：

- 通过页头「排序模式」进入拖拽态
- 支持桌面端拖拽排序，移动端触屏拖拽不在当前范围
- 排序结果自动持久化，重启后顺序保持

**图标选项**：

- 系统类：桌面、代码、文件夹、浏览器、音乐、视频、游戏、聊天、邮件、图像、终端、数据库、设计、文档、工具
- AI 类：AI应用（🤖）、AI智能（🧠）、AI助手（✨）
- 浏览器：Chrome、Firefox、Edge
- 国内常用：微信、QQ
- 开发类：GitHub、Steam

**交互逻辑**：

1. 点击应用卡片启动对应程序
2. Electron 环境下自动查找可执行文件
3. 浏览器环境下尝试使用 URL Scheme 启动
4. 支持自定义添加本地应用
5. 排序模式开启后，拖动卡片调整顺序
6. 排序模式关闭后恢复常规双击启动行为

## 交互逻辑汇总

### 导航交互

```javascript
侧边栏点击 → showPage(pageName) → 切换 page 显示 → 更新导航激活状态
                        ↓
            触发对应页面初始化函数（如果有）
```

### 数据流

```javascript
用户操作 → 修改 State → localStorage 持久化 → 重新渲染 UI
                ↓
            触发其他页面同步更新
```

### Electron 安全设置

```javascript
webPreferences: {
    nodeIntegration: false,       // 禁用 Node 集成（安全）
    contextIsolation: true,       // 启用上下文隔离（安全）
    enableRemoteModule: false,    // 禁用 remote 模块（安全）
    preload: path.join(__dirname, 'preload.js'),  // 预加载脚本
    webSecurity: false            // 开发环境禁用（生产环境视情况）
}
```

## 数据存储

### localStorage 键值表

| Key | 用途 | 数据格式 |
| ----- | ------ | --------- |
| `todos` | 待办事项数据 | JSON Array |
| `flowboard_notes` | 笔记内容 | JSON Array |
| `flowboard_events` | 日程事件 | JSON Array |
| `flowboard_repos` | GitHub 仓库缓存 | JSON Array |
| `flowboard_apps` | 应用中心配置（图标元数据+排序） | JSON Object (v2) |
| `user_profile` | 用户配置（名称） | JSON Object |
| `user_avatar` | 用户头像（Base64） | String |
| `github_username` | GitHub 用户名 | String |
| `github_token` | GitHub Token（可选） | String |
| `growth_plan_content` | 个人提升计划内容 | String |
| `growth_reminders` | 学习提醒设置 | JSON Array |
| `interview_settings` | 录音设置 | JSON Object |
| `theme` | 当前主题 | String |
| `minimizeOnLaunch` | 启动时最小化 | Boolean |
| `minimizeOnClose` | 关闭时最小化 | Boolean |
| `leetcode_submissions` | LeetCode 提交历史 | JSON Array |

### flowboard_apps v2 结构

```json
{
  "version": 2,
  "apps": [
    {
      "id": "app_1740000000000",
      "name": "VS Code",
      "path": "C:/Program Files/Microsoft VS Code/Code.exe",
      "order": 10,
      "iconMeta": {
        "mode": "preset",
        "presetClass": "fas fa-code",
        "emoji": "",
        "imageDataUrl": "",
        "bgColor": "#22c55e",
        "fgColor": "#ffffff"
      },
      "createdAt": "2026-02-25T10:00:00.000Z",
      "updatedAt": "2026-02-25T10:00:00.000Z"
    }
  ]
}
```

- `iconMeta.mode` 取值：`preset` / `emoji` / `image`
- 旧数组结构会在加载时自动迁移为 v2 对象结构

### IndexedDB 存储

| 数据库名 | 对象存储 | 用途 |
| --------- | --------- | ------ |
| `FlowBoardInterviewDB` | `recordings` | 面试录音 Blob 数据（支持大文件存储） |
