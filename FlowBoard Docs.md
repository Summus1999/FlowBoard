## 产品概述

FlowBoard 是一款面向开发者和个人用户的桌面工作台应用，采用 Electron + Web 技术栈构建，提供 glassmorphism（毛玻璃）设计风格的现代化界面。产品整合了账户管理、学习追踪、日程规划、技术刷题等多功能于一体，帮助用户高效管理数字生活。

### 设计理念

- **一站式管理**：减少在不同应用间切换的成本
- **简洁高效**：保留核心功能，避免功能冗余
- **数据本地化**：优先使用 localStorage 存储，保护隐私
- **开发者友好**：针对程序员场景优化（LeetCode、GitHub 集成）

## 核心功能详解

### 1. 我的主页

**入口**：侧边栏「我的主页」或底部 🏠 房子按钮

**功能组件**：

| 组件 | 功能描述 | 交互方式 |
|------|----------|----------|
| 账户安全概览 | 显示密码管理入口 | 点击「查看全部」进入账户管理页面 |
| 待办事项 | 显示前4个待办任务 | 点击复选框完成/取消，点击 × 删除，点击「添加待办」新增 |
| 热榜资讯 | 显示热门新闻头条 | 点击跳转外部链接 |

**交互逻辑**：
- 主页是应用启动后的默认页面
- 所有模块数据与其他页面实时同步

### 2. 账户密码管理

**入口**：主页「查看全部」或独立的账户管理页面（已移除侧边栏入口，统一从主页进入）

**预设账户**（12个国内常用账号）：
微信、支付宝、LeetCode 中国、QQ、微博、淘宝、京东、Bilibili、网易云音乐、百度、GitHub、抖音

**功能特性**：
- **分类管理**：社交/金融/工作/娱乐
- **CRUD 操作**：
  - 添加：点击「添加账户」→ 填写平台/用户名/密码/分类 → 保存
  - 编辑：点击卡片上的编辑图标 → 修改信息 → 保存
  - 删除：点击删除图标 → 确认删除
- **密码强度检测**：添加时自动计算（弱/中/强）
- **快捷操作**：点击复制图标可复制用户名/密码

**交互逻辑**：
1. 添加账户时根据分类自动分配图标和颜色
2. 密码显示/隐藏切换（眼睛图标）
3. 分类筛选按钮可快速过滤

**功能特性**：

| 财经 | 财经新闻 |

### 4. 待办事项

**入口**：主页中部 / 侧边栏「待办事项」

**功能特性**：
- **添加待办**：点击「添加待办」→ 输入内容 → 选择标签（紧急/重要/一般/日常）→ 确认
- **完成任务**：点击复选框标记完成，文字显示删除线
- **删除任务**：鼠标悬浮显示删除按钮，点击删除
- **统计同步**：未完成任务数实时同步到主页统计卡片

**标签类型**：
- 🔴 紧急（红色）
- 🟠 重要（橙色）
- 🔵 一般（蓝色）
- 🟢 日常（绿色）

### 5. 笔记记录（Markdown 编辑器）

**入口**：侧边栏「笔记记录」

**编辑器模式**：

| 模式 | 说明 | 使用场景 |
|------|------|----------|
| Markdown | 纯编辑模式 | 编写笔记 |
| 预览 | 纯渲染模式 | 查看效果 |
| 分屏 | 左编右览 | 实时对照 |

**工具栏功能**：
- 文本样式：粗体、斜体、删除线
- 标题：H1-H6
- 列表：无序列表、有序列表
- 引用块
- 代码块（支持语法高亮）
- 链接/图片插入
- 分隔线

**交互逻辑**：
1. 创建笔记：点击「新建笔记」→ 输入标题 → 选择分类 → 编辑内容
2. 自动保存：停止输入 2 秒后自动保存到 localStorage
3. 分屏模式下实时渲染 Markdown
4. 支持 Tab 键缩进
5. 快捷键：Ctrl+S 保存

### 7. 个人提升

  - 算法训练计划（数组/链表/树/图/动态规划）

- 设置提醒时间（时:分）

- 选择重复周期（周一~周日多选）

#### 7.3 学习统计
- 本周学习时长
- 本月学习时长

**交互逻辑**：
1. 导入文件后自动渲染 Markdown 内容
2. 支持任务列表复选框交互
3. 提醒到期时弹出桌面通知

### 8. 面试追踪

**入口**：侧边栏「面试追踪」

**核心功能**：

#### 8.1 录音功能
- **开始录音**：点击红色「开始录音」按钮 → 请求麦克风权限 → 开始计时
- **停止录音**：点击「停止录音」按钮 → 保存录音文件
- **录音状态**：
  - 录音中：按钮变绿色，呼吸动画效果
  - 计时器：显示录制时长（分:秒）

#### 8.2 录音设置
- **存储路径**：自定义录音文件保存目录
- **录音质量**：高质量(320kbps)/标准(128kbps)/节省空间(64kbps)
- **自动保存**：录音结束后自动保存到指定路径
- **音乐暂停**：录音时自动暂停背景音乐

#### 8.3 录音管理
- **播放录音**：点击播放按钮在线回放
- **下载录音**：下载为 .webm 格式文件
- **删除录音**：从列表和存储中删除

#### 8.4 面试辅助
- **面试笔记**：文本域记录面试要点
- **面试 Checklist**：
  - 自我介绍准备
  - 项目经验梳理
  - 技术知识点复习
  - 提问准备
  - 代码演示准备

**交互逻辑**：
1. 首次使用请求麦克风权限
2. 录音使用 MediaRecorder API
3. 录音列表实时更新
4. 支持在面试过程中随时开始/停止录音

### 9. LeetCode 刷题

**入口**：侧边栏「LeetCode」

**功能模块**：

#### 9.1 题目列表
- **数据来源**：LeetCode API（优先）/ 本地模拟数据（离线）
- **筛选条件**：难度（简单/中等/困难）、状态（全部/已解决/未解决）
- **搜索**：按题目名称/编号搜索
- **数据标识**：☁️ 官方数据 / 💾 本地数据

#### 9.2 代码编辑器
- **编辑器**：Monaco Editor（VS Code 同款）
- **支持语言**：JavaScript/Python/Java/C++/TypeScript/Go/Rust 等
- **主题**：leetcode-dark 暗色主题
- **快捷键**：Ctrl+S 运行，Ctrl+Enter 提交

#### 9.3 代码执行
- **运行代码**：本地模拟执行，显示控制台输出
- **提交代码**：模拟提交，显示通过/失败结果
- **控制台**：显示执行结果、用时、内存消耗

#### 9.4 登录功能
- Cookie 登录（支持 LeetCode 中国/国际版）
- 登录后获取真实题目数据和提交记录

**交互逻辑**：
1. 点击题目列表加载题目详情
2. 切换语言时自动加载对应代码模板
3. 支持代码重置（恢复模板）
4. 右上角新增「访问 LeetCode」外链按钮

3. 点击仓库卡片跳转 GitHub 仓库页面

## 交互逻辑汇总

### 导航交互

```
侧边栏点击 → showPage(pageName) → 切换 page 显示 → 更新导航激活状态
                        ↓
            触发对应页面初始化函数（如果有）
```

### 数据流

```
用户操作 → 修改 State → localStorage 持久化 → 重新渲染 UI
                ↓
            触发其他页面同步更新
```

### 快捷键汇总

| 快捷键 | 功能 | 适用页面 |
|--------|------|----------|
| Ctrl/Cmd + S | 保存 | 笔记、LeetCode |
| Ctrl/Cmd + Enter | 提交 | LeetCode |
| Tab | 缩进 | Markdown 编辑器 |
| ESC | 关闭弹窗 | 全局 |
