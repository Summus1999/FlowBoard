| `webSecurity` | `false` | 🔴 **危险** | 禁用同源策略 |

### 8.2 高危问题

#### 🔴 1. webSecurity 禁用
**位置**: `electron/main.js:543`
```javascript
webSecurity: false // 开发环境禁用，生产环境视情况而定
```
**风险**: 允许跨域请求，可能导致XSS攻击  
**建议**: 生产环境设置为 `true`，或通过环境变量控制

#### 🔴 2. 命令注入风险
**位置**: `electron/main.js:923-927`
```javascript
const child = spawn('"' + exePath + '"', [], {
    detached: true,
    shell: true,  // 危险
    windowsHide: false
});
```
**风险**: `exePath` 可能包含恶意字符  
**建议**: 使用 `spawn(exePath, [], { shell: false })`

#### 🔴 3. exec使用不当
**位置**: `electron/main.js:941-956`
```javascript
exec(appConfig.command, (error, stdout, stderr) => { ... });
```
**风险**: 命令注入
